# Skyrim Community Shaders - Branding Repository

This repository contains proprietary branding materials for the [Skyrim Community Shaders](https://github.com/doodlum/skyrim-community-shaders) project.

⚠️ **ALL MATERIALS ARE COPYRIGHTED** - See [LICENSE](LICENSE) for details.

## Repository Structure

- **[general-branding/](general-branding/)** - Core branding assets (logos, icons, banners, colors, fonts)
- **[nexus/](nexus/)** - Materials specifically formatted for Nexus Mods pages
- **[features/](features/)** - Screenshots, comparisons, and promotional materials for feature showcases
