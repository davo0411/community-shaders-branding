# Overview

Community Shaders feature which replaces grass shaders to add more advanced shading including Complex Grass support.

# Features

## More Advanced Grass Shading

Shades grass using vertex normals, adding directionality to lighting.
Adds subsurface scattering to the backside of grass.

## Complex Grass

Fixes UV-mapping of complex grass made for ENB.
Includes normal and specular map support.

# Installation & Requirements

1. Install [Community Shaders](https://www.nexusmods.com/skyrimspecialedition/mods/86492) and all it's requirements.
2. Install this mod below it.
3. Confirm whether or not your grass mod supports complex grass. You may need to install a complex grass update from [Vanilla Complex Grasses for ENB](https://www.nexusmods.com/skyrimspecialedition/mods/68640) (also works for CS), [Complex Grass - The Official Patch Compendium](https://www.nexusmods.com/skyrimspecialedition/mods/67304), or another source.
