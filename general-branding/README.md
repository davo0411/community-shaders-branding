# General Branding

This folder contains core branding assets for the Skyrim Community Shaders project.

## Subfolders

- **logos/** - Official project logos in various formats and sizes
- **icons/** - Application icons, favicon, and small-format branding elements. (128px .png)
- **banners/** - Banner images for social media, websites, and documentation
- **color-palettes/** - Official color schemes and brand color guidelines
- **fonts/** - Typography resources and font files used in branding
